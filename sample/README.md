# sample

Demo outputs for the three-layer workflow.

Files you should expect after running:

- `openverse-cat.jpg`: source image from Openverse
- `mail.png`: source icon from Iconify
- `cat-card.png`: edited composition with text and icon
- `cat-cutout.png`: background-removal result from `rembg`
- `cat-magick.png`: ImageMagick-based effect pass
- `ai-cat-poster.png`: AI-generated image from the prompt harness

Specs:

- `edit.card.json`
- `edit.rembg.json`
- `edit.magick.json`
- `generate.ai.json`
